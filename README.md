# InfoBar - 项目信息条工具

单行多列信息条，置顶显示项目信息。

## 安装

```bash
pip install -r requirements.txt
```

## 使用

```bash
python info_bar.py
```

## 功能

- 窗口置顶显示
- 单击拖动移动窗口位置
- 双击列文字进入编辑模式
- 拖动列之间的分隔线调整列宽
- 可自定义列内容

## 操作说明

- **单击并拖动**：移动窗口位置
- **双击列文字**：进入编辑模式，修改内容
- **编辑完成后点击其他地方**：退出编辑模式
- **拖动列分隔线**：调整列宽

## 自定义

编辑 `info_bar.py` 的 `main()` 函数中的 `columns` 列表：

```python
columns = [
    ("项目", "for_cc"),
    ("状态", "开发中"),
    ("分支", "main")
]
```
